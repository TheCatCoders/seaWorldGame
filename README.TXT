PROJECT TITLE: Sea World Game
ORGANIZATION: TheCatCoders
PURPOSE OF PROJECT: This project was created with my kids to introduce them to Object java programming.
VERSION or DATE: version 1
CODE AUTHORS: Charlotte, Olivier and Felix Roberge

HOW TO START THIS PROJECT:
    - Clone or fork the projet on git.
    - Download greenfoot from http://www.greenfoot.org/
    - Open projet on the code
    - This project work with the GitHub pull request Workflow.

USER INSTRUCTIONS FOR THE GAME:
  - Use the arrow to move the dolphin around.
  - Avoid the turtles and the lobsters.
  - Eat Star Fish to gain points.
  - Fire bullet by pressing the F key.
  - Destroying Star Fish will remove point.
  - Enjoy.
